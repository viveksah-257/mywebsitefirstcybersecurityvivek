What This Project Demonstrates (click any card for details)

OTP Authentication

One-Time Passwords add a dynamic layer — valid only for seconds/minutes. Our simulator shows generation & verification flow.

Multi-Factor Authentication (MFA)

Combining password (knowledge) + OTP (possession via mobile) dramatically reduces account takeover risk.

Banking Security

Banks use MFA/OTP to secure transactions, login, and high-risk operations — preventing fraud and unauthorized transfers.

Cyber Security Awareness

Authentication Mechanisms (click for details)

1. Password-Based Authentication

  • Traditional & most common method — user credentials
  • Vulnerable to brute-force, phishing, and weak passwords
  • Banks enforce strong password policies

2. One-Time Password (OTP)

  • Time-based or event-based, valid for single session
  • Delivered via SMS, authenticator app, or email
  • Prevents replay attacks

3. Biometric Authentication

  • Fingerprint, facial recognition, iris scan
  • Extremely secure against impersonation
  • Used in mobile banking apps

Common Cyber Attacks on Authentication (click for details)

Phishing Attack

Fake emails/websites trick users into revealing credentials, passwords, and OTPs.

Man-in-the-Middle (MITM)

Attacker secretly intercepts and alters communication between user and bank.

Brute Force Attack

Automated trial-and-error method to guess passwords, OTPs, or encryption keys.

SIM Swapping

Attacker tricks mobile carrier into transferring victim's phone number to intercept OTPs.

DDoS Attack

Distributed Denial-of-Service overwhelms systems with traffic, making services unavailable.

Malware Attack

Viruses, worms, trojans, ransomware designed to infiltrate and damage systems.

Why Banks Use Multi-Factor Authentication (click for details)

Protects from Hackers

MFA blocks 99.9% of account compromise attacks even if passwords leak.

Prevents Unauthorized Access

Attackers need physical access to second factor (phone/token) — drastically harder.

Extra Security Layer

Defense in depth: even if one factor fails, others secure the account.

Reduces Fraud in Banking

Real-time OTP verification for wire transfers, login, and high-value payments lowers fraud rate.

Educational note: modern banking apps combine push notifications, biometrics, and hardware tokens to deliver strong MFA.
